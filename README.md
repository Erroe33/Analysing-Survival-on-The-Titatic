# Analysing-Survival-on-The-Titatic

Dataset

Dataset used: Titanic Dataset

Source: Kaggle / CSV file (you can collect from my Dataset repo)

Rows: Passengers

Columns include:

Survived

Pclass

Sex

Age

Fare

Embarked

Name


**Technologies Used**

Python

Pandas

NumPy

Matplotlib

Seaborn

**Steps Performed**

Data loading and inspection

Data cleaning (handling missing values)

Feature analysis

Data visualization

Survival comparison based on different features

**Key Insights**

_Females had a higher survival rate than males

Passengers in 1st class survived more than 3rd class

Children had a better chance of survival

Higher fare passengers were more likely to survive_
